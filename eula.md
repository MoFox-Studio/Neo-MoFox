### **3. 我的个人数据是如何被处理的？**

你的隐私对我们很重要。了解数据如何流动，能帮助你更好地保护自己。

*   **工作流程**：当你与机器人互动时，你的**对话内容**、**配置信息**以及机器人**生成的回复**，会被发送给第三方的大语言模型 API 服务（例如 OpenAI、Claude、Google Gemini、本地模型等）以获得智能回复。
*   **你的明确授权**：一旦你开始使用，即表示你授权我们利用你的数据进行以下操作：
    1.  **调用外部 API**：这是机器人能与你对话的核心。你可以自由选择任何兼容 OpenAI API 格式的服务提供商，包括本地部署的模型。
    2.  **建立本地记忆系统**：这是 Neo-MoFox 的核心特性。为了让你的 AI 伙伴真正"认识"你，软件会在**你自己的设备上**创建和存储：
        - 长期记忆库（对话历史、重要时刻）
        - 知识图谱（人物关系、事件记录）
        - 个性化配置（性格特征、说话风格）
        - 向量数据库（语义检索）
        - **这些数据完全存储在本地 `data/` 目录，我们无法访问。**
    3.  **记录本地日志**：为了方便排查可能出现的技术问题，软件会在你的设备上记录运行日志（位于 `logs/` 目录）。
*   **第三方服务的风险**：我们无法控制第三方 API 提供商的服务质量、数据处理政策、稳定性或安全性。使用这些服务时，你同样受到该第三方服务条款和隐私政策的约束。我们建议你自行了解这些条款。
*   **数据控制权**：所有本地数据完全由你掌控。你可以随时查看、备份、迁移或删除这些数据。
